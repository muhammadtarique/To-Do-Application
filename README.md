# To Do Application

A simple C++ console application for managing a to-do list.

## Features

- Add multiple tasks to your to-do list
- View all tasks in your to-do list
- Remove a task from your to-do list by number
- Exit the application

## How to Compile and Run

You need a C++ compiler installed, such as g++ (part of GCC) or Visual Studio.

### Using g++ (Linux/Mac/Windows with MinGW):
```bash
g++ todoApplication.cpp -o todoApp
./todoApp
```

### Using Visual Studio (Windows):
- Open the .cpp file in Visual Studio
- Build and run the project

## Requirements

- C++ compiler supporting C++11 or later
- Standard library headers: `<iostream>`, `<vector>`, `<string>`

## Usage

Run the application and follow the menu prompts:
1. Choose to add tasks
2. View your tasks
3. Remove tasks by entering their number
4. Exit when done
